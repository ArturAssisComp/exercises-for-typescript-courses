## How to use lite-server to make the development easier

1. Init a new npm project

```bash
npm init
```

2. Install using the following command:

```bash
npm install --save-dev lite-server
```

3. Add the following script to your package.json file:

```json
...
"scripts": {
    ...
    "start": "lite-server"
}
```
